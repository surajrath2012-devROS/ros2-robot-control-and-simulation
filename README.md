# ROS 2 Industrial Warehouse Bot Simulation

An industrial logistics robot simulation built using **ROS 2 Jazzy Jalisco** and **Gazebo Sim**. This project demonstrates the integration of robotic middleware with high-fidelity physics environments.

## 🚀 Key Features
* **Industrial URDF Modeling:** A multi-tiered robot design with collision geometry and inertial properties optimized for warehouse logistics.
* **Sensor Integration:** 2D Lidar sensor data published via the `ros_gz_bridge`.
* **Systems Engineering:** Real-time communication between Gazebo Physics and ROS 2 nodes using Modern C++ infrastructure.

## 🛠️ Tech Stack
* **Language:** C++, Python, XML (URDF)
* **Robotic Framework:** ROS 2 Jazzy
* **Simulator:** Gazebo Sim (Harmonic)
* **Build System:** Colcon / CMake
